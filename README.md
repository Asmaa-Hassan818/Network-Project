# Smart City Integrated Infrastructure

## Overview

This project simulates a Smart City network infrastructure using Cisco Packet Tracer. The network is designed to connect multiple city sectors, including City Hall, Utilities, Public Safety, and the Data Center through a secure and scalable architecture.

## Technologies and Concepts

* VLSM Subnetting
* VLAN Configuration
* DHCP
* DNS
* Static Routing
* NAT (Static & PAT)
* VPN Connectivity
* Web Server Deployment
* ICMP Connectivity Testing

## Network Components

* Cisco 2911 Routers
* Cisco 2960 Switches
* PT Servers
* Serial WAN Links
* Ethernet LAN Connections

## Project Features

* Efficient IP address allocation using VLSM
* Centralized DHCP service
* DNS name resolution
* Secure VPN communication
* Internet access using NAT/PAT
* Inter-network communication through Static Routing
* VLAN segmentation for improved network management
* Web server hosting and accessibility testing
